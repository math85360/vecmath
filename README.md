# vecmath

A simple vector maths library for graphics programming.

## Quickstart

Add the following to your build.sbt:

```scala
libraryDependencies += "com.scryetek" %% "vecmath" % "0.1"
```

Or if you're using ScalaJS

```scala
libraryDependencies += "com.scryetek" %%% "vecmath" % "0.1"
```
